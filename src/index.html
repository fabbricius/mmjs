Work in progress. Play Manic Miner in your web browser! Guide Miner Willy using LEFT, RIGHT and SPACE or a joystick (using my joystick plug-in). The game graphics and layout are taken directly from the Sinclair Spectrum version. The completed project will feature a map and sprite editor.

Please let me know if you have problems getting the game to work - I'd especially like to hear from Safari users (1.0 doesn't work but newer versions should) and anyone experiencing slow level loading (where the screen is built up one block at a time rather than all at once).
